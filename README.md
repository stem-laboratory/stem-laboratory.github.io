# STEM-лабораторія педагогічних практик

Електронна збірка STEM-проєктів для вчителів інформатики Запорізької області.

## 🌐 Адреса сайту
`https://stem-laboratory.github.io`

## 📁 Структура проєкту
```
stem-laboratory/
├── index.html          ← головна сторінка (каталог + пошук + фільтри)
├── photos/             ← 6 фото для каруселі (photo1.jpg … photo6.jpg)
├── css/
│   └── style.css       ← всі спільні стилі
├── js/
│   ├── main.js         ← карусель, фільтри, пошук, анімації
│   └── projects.js     ← дані всіх занять
├── projects/
│   ├── session-1.html  ← Вектор розвитку + STEM-радар
│   ├── session-2.html  ← Розумний будинок + рефлексія (16 відповідей)
│   ├── session-3.html  ← Робототехніка + рефлексія (14 відповідей)
│   ├── session-4.html  ← Codey Rocky + рефлексія (9 відповідей)
│   └── session-5.html  ← 3D-друк + AI + рефлексія (10 відповідей)
├── assets/
│   └── icons/
└── README.md
```

## 📸 Додавання фотографій
Помістіть у папку `photos/` шість файлів:
```
photo1.jpg … photo6.jpg
```
Рекомендований розмір: **1920×900 px**, формат JPG. Карусель підтягне їх автоматично.

## 🚀 Розгортання на GitHub Pages

### Крок 1 — Реєстрація
- Зареєструйтесь на [github.com](https://github.com)
- Username: **`stem-lab`** → адреса буде `https://stem-lab.github.io`

### Крок 2 — Репозиторій
- **New repository** → назва: `stem-lab.github.io`
- Visibility: **Public**

### Крок 3 — Завантаження файлів
- **Add file → Upload files** → перетягніть усі файли зі збереженням структури папок
- **Commit changes**

GitHub Pages вмикається автоматично для репозиторіїв з назвою `username.github.io`.

## ➕ Додавання нового заняття

1. Скопіюйте `projects/session-5.html` → `projects/session-6.html`
2. Замініть контент (назва, опис, кроки, рефлексія)
3. Додайте картку в `index.html` (за шаблоном існуючих)
4. Додайте запис у `js/projects.js`
5. Оновіть посилання «Наступне» в session-5.html

## 📊 Дані рефлексій
Інфографіки побудовані на реальних даних із 5 файлів Google Forms:

- Робототехніка: 14 відповідей, Tinkercad 4.5/5
- Codey Rocky: 9 відповідей, зрозумілість 5.0/5
