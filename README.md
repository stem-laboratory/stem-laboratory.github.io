## 🚀 Розгортання на GitHub Pages

### Крок 1 — Створіть репозиторій

1. Увійдіть на [github.com](https://github.com)
2. Натисніть **New repository**
3. Назва: наприклад `stem-lab` або `stem-projects`
4. Видимість: **Public**
5. Натисніть **Create repository**

### Крок 2 — Завантажте файли

**Варіант А — через браузер (простіше):**
1. У репозиторії натисніть **Add file → Upload files**
2. Перетягніть усі файли з папки `stem-lab/`
3. Натисніть **Commit changes**

**Варіант Б — через Git:**
```bash
git init
git add .
git commit -m "initial: STEM lab projects site"
git branch -M main
git remote add origin https://github.com/USERNAME/stem-lab.git
git push -u origin main
```

### Крок 3 — Увімкніть GitHub Pages

1. Перейдіть у **Settings** репозиторію
2. Знайдіть розділ **Pages** (ліве меню)
3. Source: **Deploy from a branch**
4. Branch: **main** / **(root)**
5. Натисніть **Save**

Через 1–2 хвилини сайт буде доступний. Посилання з'явиться у розділі Pages.

## ➕ Як додати нове заняття

### 1. Скопіюйте файл шаблону

Скопіюйте будь-який файл `session-N.html` і перейменуйте, наприклад `session-6.html`.

### 2. Відредагуйте вміст

Замініть:
- заголовок і опис заняття
- цілі, кроки, інструменти
- посилання на матеріали
- обладнання та предмети

### 3. Додайте картку на головну

Відкрийте `index.html`, знайдіть блок `<!-- Заняття 1 -->` як зразок і додайте нову картку за тим самим шаблоном перед `</div>` закриттям `cardsGrid`.

### 4. Оновіть навігацію між заняттями

У попередньому занятті змініть посилання «Наступне» на `session-6.html`.
